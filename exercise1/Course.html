CSS

CSS, also known as Cascading Style Sheets is a markup language that allows for styling of text & Webpage elements, CSS is versatile and has a wide range of capabilities that allow for the simple modification of text and elements, to far more advanced techniques that can mimic some photo rendering software.

Used the right way, CSS can add a professional look & feel to almost any Webpage.
